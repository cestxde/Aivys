# Branding Guidelines for Aivys Player

This document outlines the rules for using the visual identity of the Aivys Player project.

## Ownership
The "Aivys Player" name, logo, and all related brand assets are the intellectual property of the project author. Graphical assets in the `src-tauri/icons`, `src-tauri/branding` and `src/assets` directories are provided under the **CC BY-NC-ND 4.0** license.

## Allowed Use ✅
* Using the logo to link back to this repository.
* Using the logo in articles, video reviews, or news coverage about Aivys Player.
* Including the logo in software directories or catalogs.

## Prohibited Use ❌
* Using the logo in your own software products (forks) without significant visual changes.
* Modifying the colors, proportions, or geometry of the logo.
* Using the name or logo for commercial profit without prior consent.

---
For any questions regarding the use of these assets, please open a GitHub Issue.
